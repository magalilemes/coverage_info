     150             :  *   Req2
     151             :  *     Readers must see that the ringbuffer is initially empty.
     152             :  *
     153             :  *   Req3
     154             :  *     The first record reserved by a writer is assigned sequence number 0.
     155             :  *
     156             :  * To satisfy Req1, the tail initially points to a descriptor that is

     159             :  *
